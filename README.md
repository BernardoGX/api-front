# API Front-End

A simple and lightweight front-end interface for building and testing API requests.

## Features

- **Input Field**: Enter your API endpoint or data
- **Three Action Buttons**: Quickly execute common API operations
- **Response Display**: View results in a dedicated text area

## Project Structure

```
api-front/
├── index.html      # Main HTML structure
├── style.css       # Styling and layout
├── script.js       # JavaScript functionality
└── README.md       # Project documentation
```

## Getting Started

1. Open `index.html` in your web browser
2. Enter your API endpoint or request details in the input field
3. Click one of the three action buttons to execute your request
4. View the response in the text area below

## Usage

- **Input Field**: Enter API URLs, endpoints, or request parameters
- **Buttons**: Perform actions such as GET, POST, or custom operations
- **Text Area**: Displays responses, results, or error messages

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
